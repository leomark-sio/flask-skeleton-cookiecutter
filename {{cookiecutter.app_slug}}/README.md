## Quick Start

Install Cookiecutter globally:

```sh
$ pip install cookiecutter
```

Generate the boilerplate:

```sh
$ cookiecutter https://github.com/leomark-sio/cookiecutter-flask-skeleton.git
```

Review the setup guides to configure the app:

1. [setup-with-docker.md](setup-with-docker.md)
2. [setup-without-docker.md](setup-without-docker.md)
